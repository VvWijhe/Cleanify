
## Dependencies
- [Restbed](https://github.com/Corvusoft/restbed) (included as submodule)
- Poco libraries
- Boost libraries

## Build
- `$ git clone https://github.com/VvWijhe/Cleanify`
- `$ cd Cleanify`
- `$ git submodule update --init --recursive`
- `$ cd RoombaController`
- `$ mkdir bin`
- `$ cd RoombaController/bin`
- `$ cmake ..`

## Run
- `$ ./RoombaController`
